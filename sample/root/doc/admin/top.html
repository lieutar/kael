管理者トップ  

こちらは Reds Package Solution 「ウェブマスター・コーポレートパック」の管理サイトです
左のメニューより管理したい部分をお選びください。

  • ニュースヘッドライン／インフォメーション管理

    発信したいニュース情報の「新規追加」「編集／更新」「削除」が行えます。
    貴社サイトのトップページ、インフォメーションに反映されます。

  • 業務内容 FAQ 管理

    業務上よくあるお問い合わせに対してサイト上で回答を掲載しておくことができます。
    “質問と答え”のセットの「新規追加」「編集／更新」「削除」「順序変更」が行えます。

  • 採用情報管理

    アルバイト・パートから正社員まで採用の告知ができます。
    採用情報の 「新規追加」「編集／更新」「削除」が行えます。
    また掲載された情報に対して送られてきた応募のデータ（ＣＳＶ形式）を一括ダウンロードすることができます。

  • お問い合わせ管理

    お問い合わせフォームより送信された内容を（ＣＳＶ形式）を一括ダウンロードすることができます。
